# Iconos de Bancos

Esta carpeta contiene los iconos de los diferentes bancos y fuentes de transacciones.

## Iconos Necesarios

Descarga los siguientes iconos en formato PNG (preferiblemente 512x512 o 1024x1024):

### Bancos Ecuatorianos:
- `banco_pichincha.png` - Banco Pichincha (rojo/blanco)
- `produbanco.png` - Produbanco (azul)
- `banco_guayaquil.png` - Banco Guayaquil (azul/amarillo)
- `banco_pacifico.png` - Banco Pacífico (azul)

### Procesadores de Pago:
- `deuna.png` - Deuna (morado/blanco)

### Tarjetas de Crédito:
- `diners_club.png` - Diners Club (azul/blanco)

### Default/Manual:
- `manual.png` - Para transacciones manuales (ícono de billete o moneda)
- `default.png` - Para bancos no reconocidos (ícono de banco genérico)

## Formato Recomendado:
- **Formato**: PNG con transparencia
- **Tamaño**: 512x512 px o superior
- **Estilo**: Logo oficial del banco, preferiblemente circular o con bordes redondeados
- **Fondo**: Transparente

## Fuentes sugeridas:
1. Sitios web oficiales de los bancos
2. Brandfetch.com
3. Logos descargables de las apps oficiales

## Nombres de archivo (exactos):
```
banco_pichincha.png
produbanco.png
banco_guayaquil.png
banco_pacifico.png
deuna.png
diners_club.png
manual.png
default.png
```

Una vez descargados los iconos, colócalos en esta carpeta.
